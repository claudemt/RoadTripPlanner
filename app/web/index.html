配置高德地图

填写高德 Web JS API Key 和安全密钥后加载地图。

  1. 打开 高德开放平台控制台
  2. 创建「Web端(JS API)」应用，获取 Key
  3. 在安全密钥中设置并复制 securityJsCode
  4. 将本机域名/地址加入白名单：127.0.0.1、localhost
配置会写入浏览器，并同步到 data/config/local.env（gitignore，不会进 GitHub）。
