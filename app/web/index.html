配置高德地图

填写高德 Web JS API Key 和安全密钥后加载地图。

  1. 打开 高德开放平台控制台
  2. 创建「Web端(JS API)」应用，获取 Key
  3. 在安全密钥中设置并复制 securityJsCode
  4. 将本地域名或网站正式域名加入高德白名单
配置保存在服务器本机，仅通过 Caddy 代理访问。
